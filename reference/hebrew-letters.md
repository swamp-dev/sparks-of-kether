# The 22 Hebrew Letters

Each letter *is* a path on the Tree. The **Sepher Yetzirah** (Book of
Formation, ~2nd century) classifies the 22 letters into three groups, each
carrying a different cosmic attribution.

## The three classifications

### 3 Mothers — the Elements

The primordial forces. Each is a horizontal or world-dividing path on the Tree.

| Letter | Element | Path | Role |
|---|---|---|---|
| **Aleph** (א) | Air | 11 | Breath before speech. The Fool. |
| **Mem** (מ) | Water | 23 | Yielding, unconscious depth. The Hanged Man. |
| **Shin** (ש) | Fire | 31 | Spirit, divine breath, awakening. Judgement. |

## A note on gematria

Each letter carries a numerical value (see Master table). Hebrew words can be
summed — "gematria" — to reveal hidden equivalences (e.g., *ahavah* "love" =
13 = *echad* "one"). The game doesn't require gematria to play, but the
letter values double as the **card values** for mechanics that need numbers
(sums, sequences, ties).
